# Sample AI - Frontier AI Landing Page

A modern, responsive landing page for Sample AI, featuring elegant animations, customer testimonials, and a beautiful beige-toned design.

## ✨ Features

- **Responsive Design** - Fully optimized for all devices (mobile, tablet, desktop)
- **Smooth Animations** - Powered by Framer Motion for elegant page transitions
- **Modern UI** - Dark beige customer cards with glassmorphism effects
- **Interactive Elements** - Hover effects, logo scroll, and customer carousel
- **Grid Backgrounds** - Subtle geometric patterns for visual depth
- **Mobile Navigation** - Hamburger menu for smaller screens

## 🛠️ Tech Stack

- **React 19** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast development and build
- **Tailwind CSS 4** - Utility-first styling
- **Framer Motion** - Animation library

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/atharvavdeo/Ladning_Pages.git

# Navigate to project directory
cd landing1

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🏗️ Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Components

- **LandingPage** - Hero section with mountain backdrop and chat input
- **CustomersPage** - Customer testimonials with bento grid layout
- **Navigation** - Responsive navbar with mobile hamburger menu
- **InfiniteLogoScroll** - Animated company logo carousel
- **CustomerCarousel** - Interactive customer review slider

## 🎯 Key Design Features

### Mobile Optimization
- Responsive typography scaling
